# /setup — Configure aria-knowledge

Configure the aria-knowledge plugin: create or validate the knowledge folder, check dependencies, set audit cadences, and write config to `~/.gemini/antigravity/aria-knowledge.local.md`.

## Steps

Invoke the aria-knowledge **`setup`** skill — its `SKILL.md` lives at `~/.gemini/config/plugins/aria-knowledge/skills/setup/SKILL.md` (or `.agents/plugins/aria-knowledge/skills/setup/SKILL.md` for workspace install). Follow the 8+ steps verbatim: config detection, knowledge folder setup, file diffing, dependency check, cadence configuration, project tier scaffolding, and self-validation.

Safe to re-run at any time — only touches what needs updating.
